72-Hour
Incident Playbook.

A deterministic response framework for autonomous system failures. Truth precedes correction.

Hour 01 // Identification

Integrity Anomaly Detection

System flags an artifact mismatch or a signing failure. The "Fail-Open" status of the inference engine is confirmed while the Integrity Layer enters "Audit Isolation."

Hour 02-12 // Preservation

Artifact Bundling

All artifacts associated with the incident window are cryptographically bundled. Public keys are prepared for institutional review. No data is modified until the bundle is sealed.

Hour 12-48 // Verification

Third-Party Handover

The sealed bundle is provided to the Authorized Institution. Independent verification is performed using the standalone CLI tool to confirm the chain of custody remains intact.

Hour 48-72 // Reporting

Post-Incident Disclosure

A formal integrity report is generated. The institution provides the final adjudication based on the deterministic records preserved by NADIR.
