Controlled Review Requests

Institutions seeking technical evaluation may request controlled review access. Controlled review may include:

  • Technical Requirements Document access
  • Capture policy schema review
  • Verification contract review
  • Reference benchmark discussion
  • Architecture Q&A session

Controlled review does not constitute license or deployment authorization.

Deployment Inquiries

Organizations exploring pilot integration or phased deployment may submit deployment inquiries. Initial discussions typically address:

  • Deployment architecture posture
  • Capture policy configuration
  • Infrastructure compatibility
  • Key custody integration
  • Disclosure tier requirements

Deployment terms are defined through written agreement.
