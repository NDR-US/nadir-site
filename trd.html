Technical Requirements Document

Outline & Scope

Section 01

Host Environment Compatibility

Defines the runtime constraints for NADIR sidecar/wrapper integration. Includes CPU/Memory overhead targets (<1.5% inference latency impact) and network topology requirements.

Section 02

Canonicalization Protocol

Detailed specification of RFC 8785 implementation. Field ordering, character encoding (UTF-8), and whitespace normalization procedures to ensure cross-platform hash consistency.

Section 03

Cryptographic Handshake

Requirement for PKCS#11 or KMIP compatibility for HSM-based signing. Specifies P-256 (ECDSA) curve requirements and non-custodial key-access policies.

Section 04

Artifact Persistence & Egress

Schema for the sealed `.nadir` artifact. Defines metadata headers, payload structure, and authenticated egress points for institutional retrieval.
