Verification Substrate.

A non-intrusive evidentiary layer that operates alongside inference cycles, producing immutable decision-state records without mutating the core model logic.

01. Boundary Attachment

Integration occurs at the inference boundary via sidecar, gateway, or API wrapper. This ensures NADIR observes events without modifying weights or training pipelines.

02. Non-Custodial Security

Signing keys are generated and retained within client-controlled environments (HSM or TEE). NADIR never possesses or transmits private keys.
